# ApplicationProvider

ApplicationProvider is the root component of a Kittsune application. It provides Eva Design System styles, theme variables, and icon packs to all descendant components. Every Kittsune app must wrap its root component tree with `ApplicationProvider`.

## Import

```tsx
import { ApplicationProvider } from '@kittsune/components';
```

## Basic Usage

```tsx
import React from 'react';
import * as eva from '@eva-design/eva';
import { ApplicationProvider, Layout, Text } from '@kittsune/components';

const App = () => (
  <ApplicationProvider {...eva} theme={eva.light}>
    <Layout style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text category="h1">Welcome to Kittsune</Text>
    </Layout>
  </ApplicationProvider>
);

export default App;
```

## Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `mapping` | `SchemaType` | - | Eva mapping configuration object. |
| `theme` | `ThemeType` | - | Eva theme object (e.g., `eva.light` or `eva.dark`). |
| `customMapping` | `CustomSchemaType` | `undefined` | Custom mapping overrides for component styles. |
| `children` | `ReactNode` | - | The application component tree. |
